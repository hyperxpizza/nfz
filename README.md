# Fake polish healthcare covid test app
Basically a copy of meatspin.fr LOL

# Build
```bash
docker build -t flask-python-covidapp:latest .
```

or with docker-compose
```
docker-compose buld
```

# Run
```bash
docker run -d 5000:5000 flask-python-covidapp
```
or with docker-compose

```bash
docker-compose up -d
```
